# Release scripts

## start.sh

To start the release in all repositories, execute:

```bash
./start.sh
```

### Example

```bash
/usr/bin/env bash ~/Projects/github.com/Privatix/privatix/scripts/release/start.sh
Enter the release version: 0.14.0
```
